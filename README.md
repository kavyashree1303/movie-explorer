# 🎬 Movie Explorer

Movie Explorer is a dynamic React + Vite web app that allows users to search for movies using the [OMDb API](https://www.omdbapi.com/), view poster cards, and explore full movie details in a clean and responsive UI.

## 🛠️ Setup & Run Instructions

### ✅ Prerequisites
- Node.js and npm installed
- OMDb API key (get one for free from [https://omdbapi.com/apikey.aspx](https://omdbapi.com/apikey.aspx))

### 🧑‍💻 Local Development

```bash
# Clone the repo
git clone https://github.com/your-username/movie-explorer.git
cd movie-explorer

# Install dependencies
npm install

# Add your OMDb API key
# Open src/api.js and replace 'your_api_key_here' with your actual key

# Start the development server
npm run dev
